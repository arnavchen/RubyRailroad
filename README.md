An animated Rails demo featuring a train traveling across the screen. Each car represents a core Rails concept.

## Setup

```bash
bundle install
rails server
```

Open http://localhost:3000

## Features

- Animated train with 6 cars (Engine, MVC, ActiveRecord, Routes, Controllers, Views)
- Speed control slider
- Hover to pause
- Fully responsive

## Stack

- Rails 7.2
- Pure CSS animations
- Minimal JavaScript
